接单服务器操作指南

本页面帮助你快速了解如何检查和管理 Mobiel Server（接单服务器）在 Windows 电脑上的运行状态。

1. 服务器自动启动说明

  • 接单服务器会随电脑开机自动在后台运行。
  • 你无需手动点击桌面上的 Mobiel Server 图标来启动服务器。
  • 如遇特殊情况（如服务器未自动运行），可手动点击桌面图标尝试启动。

2. 检查服务器是否在后台运行

  1. 在任务栏空白处右键，选择 Taakbeheer（任务管理器）。
  2. 在进程列表中，找到 Boxmeer-new.exe，显示在“后台进程”或“应用”下。
  3. 如图所示，出现该进程即为正常运行。
  4. 如未看到该进程，可重启电脑或手动点击桌面 Mobiel Server 图标。

常见问题

  • 如遇服务器无法启动、进程不在列表、或下单异常，请联系技术支持。
  • 建议在餐厅运营期间不要随意关闭 Mobiel Server 相关进程。
  • 如发生听到进单的播报声音，但打印机未出单，请检查2号主机(餐厅中间那部电脑)是否已经开启。
